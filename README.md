# The Last Withdrawer

## Overview
This is a turn-based survival game developed **100% in Java**. All game mechanics, logic, and UI have been implemented using Java.

## Technologies Used
- **Java 100%**
    - No other programming languages or frameworks are used, except for Java itself.
    - JavaFX for graphical user interface (GUI)
    - JUnit for testing

## Getting Started
To run the game, ensure you have Java (JDK) installed. You can download it from [Oracle's official website](https://www.oracle.com/java/technologies/javase-jdk14-downloads.html).

### Running the Game
1. Download the project repository.
2. Open a terminal/command prompt and navigate to the project directory.
3. Run the following command:

    ```bash
    java -jar The_Last_Withdrawer-1.0.jar
    ```

4. Enjoy the game!

## Features
- **Turn-based mechanics**
- **Multiple zombie types** with different abilities
- **Player inventory** and weapons
